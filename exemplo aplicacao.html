Aprenda os princípios básicos para confecções de um site em HTML para que você possa fazer o seu primeiro site.

Motivação

HTML (abreviação de Hypertext markup Language) é uma linguagem de marcação utilizada na estruturação utilizada na estruturação de págias web. Sua sintaxe é bastante simples e, assim como XLM é baseada em tags, que representam os diversos elementos de uma página, como imagnes e links.
